# Pet-Adoption-Analytics-Dashboard-Power-BI-
The Pet Adoption Analytics Dashboard is an interactive Power BI project designed to transform raw pet adoption data into meaningful, actionable insights.  This dashboard enables users to explore adoption trends, understand pet characteristics, and make data-driven decisions through dynamic filtering and intuitive visualizations.

🎯 Problem Statement

Pet adoption centers often struggle to:                                                                                      
Track adoption trends effectively                                                                                            
Understand which pets are more likely to be adopted                                                                          
Analyze factors influencing adoption decisions                                                                               

This dashboard solves these challenges by providing a centralized and interactive analytical view of pet adoption data.

🚀 Key Features                                                                                                             
🔢 KPI Metrics                                                                                                              
Total Available Pets                                                                                                         
Total Adopted Pets                                                                                                           
Average Adoption Fee                                                                                                         

These KPIs provide a quick snapshot of overall adoption performance.

📊 Visualizations & Insights                                                                                                
Adoption Status Breakdown                                                                                                    
→ Clear comparison of adopted vs available pets                                                                              
Vaccination Analysis                                                                                                         
→ Tracks vaccinated vs non-vaccinated pets                                                                                   
Health Condition Insights                                                                                                    
→ Helps understand pet health distribution                                                                                   
Size Distribution                                                                                                            
→ Identifies adoption trends by pet size                                                                                     
Category Analysis                                                                                                            
→ Dogs, Cats, Birds, Rabbits segmentation 

🎛️ Interactive Filtering System                                                                                             

The dashboard includes dynamic slicers for:

Breed                                                                                                                        
Color                                                                                                                        
Size                                                                                                                         
Vaccination Status                                                                                                           

Users can drill down into specific segments for deeper analysis.
